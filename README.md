计算摩尔质量（相对原子/分子质量）的python程序。
# 化学式相对质量计算器

一个简单易用的命令行工具，用于根据化学式计算相对分子质量（或相对原子质量）。适合中学生、大学生及化学爱好者快速验证化学式的质量。

### 功能特点

- 支持常见元素（约 80 种）的相对原子质量（中学化学常用近似值）。
- 解析简单化学式（如 `H2SO4`、`CaCO3`、`NaOH`），自动提取元素符号和下标。
- 输出计算过程，显示每个元素的质量贡献。
- 对于单原子分子（如 `Fe`、`O`），直接返回整数或浮点数（无多余过程）。
- 友好的交互式命令行界面，支持连续输入，输入 `q` 退出。

### 运行环境

- Python 3.6 及以上版本（使用了类型注解 `list[tuple[str, int]]`）。

### 启动程序

在终端中执行：

```bash
python molar_mass.py
```
（或下载releases中的可执行文件（exe）运行）

输入化学式
程序启动后，根据提示输入化学式（不区分大小写，但元素符号需符合规范，如 Na、Cl）。示例：

请输入化学式 (输入 q 退出): H2O
  计算过程: H(1)×2 + O(16) = 18.0
  H2O 的相对质量 = 18
退出程序
输入 q 或 Q 即可退出。

支持的化学式格式
单元素：Fe、O、He 等。

多元素组合：NaCl、H2SO4、CaCO3、NaOH 等。

下标为数字（整数），如 H2O、C6H12O6。

元素符号必须首字母大写，第二个字母小写（如 Cl、Mg）。

⚠️ 注意：本程序目前 不支持 含括号、水合物（如 CuSO4·5H2O）、离子基团（如 (NH4)2SO4）等复杂结构。仅适用于简单线性化学式。

示例
输入化学式	输出结果（相对质量）
H2O	18
H2SO4	98
CaCO3	100
NaOH	40
NaCl	58.5
Fe	56
O2	32
### 元素数据
程序中预置了常见元素的相对原子质量（近似值），来源为中学化学常用数据。若需要添加或修改元素，可编辑 ATOMIC_MASS 字典。

### 限制与改进方向
不支持括号：如 (NH4)2SO4 无法正确解析。

不支持水合物：如 CuSO4·5H2O。

不支持带电荷的离子式：如 SO4^2-。

原子质量值均为近似值，如需高精度请自行修改字典。

未来可考虑添加对括号、水合物和更复杂结构的支持。
